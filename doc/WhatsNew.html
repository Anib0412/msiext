    • ODBCExecute and ODBCExecuteFile data source extension supports parsed SQL, error commands (:On Error) and file inclusions with new Type (SqlServer, Oracle) and BasePath options.

    • Added ODBC_Execute_Binary immediate custom action.
    • ODBC_Execute and ODBC_Execute_Binary custom actions return ODBC_SQL_MESSAGES, out-of-band SQL status messages.
    • ODBQ: added --flattenonly, --rawoutput and short versions -f (--file) and -o (--outputfile).
